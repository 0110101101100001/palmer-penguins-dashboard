O **DataLens Dashboard** é um ecossistema completo de análise e predição baseado no dataset *Palmer Penguins*. Desenvolvido com uma estética de terminal e IDE, o projeto integra processamento de dados, visualização estatística e inteligência artificial em uma interface reativa.

## 🚀 Funcionalidades Principais

### 📊 Pipeline de Analytics
- **Tour do Projeto:** 11 slides detalhando desde a ingestão até os insights do modelo.
- **Analisador Dinâmico:** Filtros em tempo real para métricas, ilhas e sexo dos pinguins.
- **Engenharia de Atributos:** Visualização moderna do fluxo de tratamento de dados (Scaling, Encoding, Dummies).

### 🤖 Inteligência Artificial
- **Motor de Inferência:** Predição de espécie baseada em medidas de nadadeira e bico utilizando algoritmos de Machine Learning.
- **Terminal de Logs:** Sistema de logs em tempo real que monitora todas as ações do usuário e processos de boot.

### 🎨 UI/UX Moderna
- **Dual-Mode:** Troca dinâmica entre o tema **Dracula (Dark)** e **Clean (Light)**.
- **Estética Hacker:** Uso da tipografia *JetBrains Mono* e elementos de design inspirados em editores de código.

## 🛠️ Stack Tecnológica

* **Backend:** Python e Micro-framework Flask.
* **Data Science:** Pandas para manipulação, NumPy para cálculos estatísticos e Scikit-Learn para o modelo preditivo.
* **Frontend:** HTML5, CSS3 (com Glassmorphism e Variáveis Dinâmicas) e JavaScript (Vanilla).
* **Visualização:** Plotly.js para gráficos interativos e integração com mapas táticos.
