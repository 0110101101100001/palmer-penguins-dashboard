O **Palmer Penguins Dashboard** é um ecossistema completo de Ciência de Dados voltado para a análise e predição das espécies de pinguins do Arquipélago Palmer, na Antártica. O projeto integra desde o processamento de dados brutos até a inferência em tempo real utilizando Inteligência Artificial.

## 🚀 Funcionalidades

### 📊 Análise Exploratória (EDA)
- **Pipeline de Analytics:** Visualização passo a passo do tratamento dos dados.
- **Gráficos Dinâmicos:** Analisador interativo para cruzamento de métricas (massa corporal, nadadeira, bico).
- **Mapa Tático:** Localização geográfica das estações de pesquisa (Biscoe, Dream e Torgersen).

### ⚙️ Engenharia de Dados
- **Sanitização Automatizada:** Módulo exclusivo para limpeza de arquivos CSV (remoção de NaNs e normalização).
- **Dicionário de Dados:** Documentação técnica completa de todas as variáveis do dataset.

### 🤖 Inteligência Artificial
- **Motor de Inferência:** Predição de espécie baseada em medidas morfológicas.
- **Interface Dual-Mode:** Troca dinâmica entre modo escuro (Dracula Theme) e modo claro (Clean Design).

## 🛠️ Tecnologias Utilizadas

- **Backend:** Python & Flask
- **Frontend:** HTML5, CSS3 (Variáveis Dinâmicas), JavaScript (Vanilla)
- **Data Science:** Pandas, NumPy, Scikit-Learn
- **Visualização:** Plotly.js & Seaborn
- **Design:** Dracula Theme Palette & JetBrains Mono Font
